# Ssonic

**Ssonic** to nowoczesny i responsywny szablon galerii online, przygotowany z myślą o prostym prezentowaniu zdjęć oraz portfolio w przeglądarce.

Projekt został stworzony przy użyciu **Vue 3** oraz **Vite**. Szablon można pobrać, uruchomić lokalnie i dostosować do własnych potrzeb.

## Demo

Demo projektu dostępne jest pod adresem:

```txt
https://ssonic.progra24.pl/
```

## Repozytorium

Kod źródłowy projektu:

```txt
https://github.com/dresnok/ssonic
```

## Funkcje projektu

* responsywna galeria zdjęć,
* automatyczne generowanie miniaturek,
* tryb podglądu fullscreen,
* pokaz slajdów,
* lekka i przejrzysta struktura projektu,
* brak logowania i kont użytkowników,
* możliwość łatwego dostosowania szablonu do własnych potrzeb.

## Technologie

Projekt wykorzystuje:

* Vue 3,

## Jak uruchomić projekt lokalnie

1. **Sklonuj repozytorium:**

```bash
git clone https://github.com/dresnok/ssonic.git
```

2. **Wejdź do katalogu projektu:**

```bash
cd ssonic
```

3. **Zainstaluj wymagane biblioteki:**

```bash
npm install
```

4. **Uruchom serwer deweloperski:**

```bash
npm run dev
```

Po uruchomieniu tej komendy projekt powinien być dostępny w przeglądarce pod adresem:

```txt
http://localhost:5173/
```

## Budowanie wersji produkcyjnej

Aby wygenerować wersję produkcyjną projektu, użyj komendy:

```bash
npm run build
```

Po zakończeniu budowania gotowe pliki znajdziesz w katalogu:

```txt
dist
```

## Podgląd wersji produkcyjnej

Po zbudowaniu projektu możesz sprawdzić wersję produkcyjną lokalnie:

```bash
npm run preview
```

## Dostosowanie projektu

Szablon możesz dostosować do własnych potrzeb, modyfikując pliki znajdujące się w katalogu `src`.

Najczęściej edytowane elementy to:

* wygląd galerii,
* układ strony,
* style CSS,
* zdjęcia i multimedia,
* teksty opisowe,
* konfiguracja komponentów Vue.
